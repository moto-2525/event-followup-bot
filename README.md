# event-followup-bot

Google Apps Script を使って、  
**イベント終了後アンケートの回答内容に応じたお礼メールを自動送信**するスクリプトです。

Google フォーム → Google スプレッドシート → Gmail  
という流れをシンプルに自動化しています。

---

## 概要

- Google フォームの最新回答を取得
- 回答者の満足度に応じてメール本文を分岐
- 個別にカスタマイズしたお礼メールを自動送信

イベント運営後のフォローアップを想定した、  
実務でも使いやすい構成です。

---

## 主な機能

- 最新のフォーム回答を自動取得
- 満足度（非常に満足 / 満足 / 普通 / 不満足）に応じた本文切り替え
- Gmail を使った自動メール送信
- 改行・文章構成を考慮した読みやすいメール本文

---

## 使用サービス / API

- Google Apps Script
- Google Spreadsheet
- GmailApp

※ Advanced Google Services は使用していません

---

## セットアップ手順

1. Google フォームと連携したスプレッドシートを作成
2. Apps Script プロジェクトを新規作成
3. 本リポジトリのコードを貼り付け
4. シート名をフォームの回答シート名に合わせる  
   （例: `フォームの回答 1`）
5. 必要に応じてトリガーを設定  
   （フォーム送信時など）

---

## ファイル構成

```text

```

## 補足

- メール送信先はフォーム回答内のメールアドレスを使用しています
- 学習・個人利用・小規模イベントでの使用を想定しています
- 文面は用途に応じて自由にカスタマイズ可能です
